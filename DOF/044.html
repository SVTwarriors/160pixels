Chain of Events: Not part of a Chain

 

Canceled by / No effect after Resolution of: None

 

  
Card Text: Target any country where your opponent has Power. Remove 2 opponent SPs and make 1 Support Check in the country using the Ops value of this card. Your opponent may cancel the Support Check by discarding a 3 Ops card.

   

Democrat play: The Inflationary Currency event is much like a weaker version of the Gorbachev Charms the West event through it’s multiple limitations. Not only is it less flexible because the whole event (both the SP removal and the Support Check) needs to be conducted in the same country and this has to be a country where the player’s Opponent Holds Power. The SC can also be canceled by an opponent discard of a 3 Ops card if he so chooses. Because the countries where the Communist player still Holds Power are usually the ones where most of the action is anyway this isn’t such a big deal for the Democrat player. The discard can work well with Democrat events that reduce the Communist player’s hand size (e.g. the Presidential Visit event, the Kiss of Death event and the Goodbye Lenin! event) the Inflationary Currency event is a weaker event for the Communist player than it is for the Democrat player. (The Communist player will also be more likely to use the Common European Home event early thereby making himself vulnerable to further hand size reductions.) This can result in back-to-back Action Rounds. Be sure the Communist player can’t increase his hand with the TST People’s Daily Denounces Protesters spot Bonus or the Honecker event). Otherwise if the Democrat player has a good Support Check target trigger the Inflationary Currency event near the end of the Turn when the Communist player could’ve run out of 3 Ops cards or if the Democrat player has counted that the Communist player isn’t holding any or the discard would force him to play a Democrat card the Communist player would prefer to hold.

 

   

Communist play: Because the Inflationary Currency event can only be used by the Communist player in a country where the Democrat player Holds Power and the Democrat player can combine it with many more Democrat events that reduce the Communist player’s hand size (e.g. the Presidential Visit event, the Kiss of Death event and the Goodbye Lenin! event) the the Inflationary Currency event is a weaker event for the Communist player than it is for the Democrat player. (The Communist player will also be more likely to use the Common European Home event early thereby making himself vulnerable to further hand size reductions.) Therefore the Communist player is more likely to play the Inflationary Currency card for 3 Ops unless he has a good Support Check target and therefore thinks he might trick the Democrat player into a double discard by first allowing the Democrat player to discard a 3 Ops card on the the Inflationary Currency event and then following it up with either the Brought in for Questioning event or the Deutsche Marks event.
