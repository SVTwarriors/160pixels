Player (Card's Faction): The cards's faction, either Communist (C), Democrat (D) or Neutral (N).

Period (1989): The Period the event occured in, either Early Year (EY), Middle Year (MY) or Late Year (LY)

Ops (Operation Points): The normal Operation Points value of the card. (0-4 Ops)

Type (Event Type): The type of Event, either One-time, Recurring, and/or Red (for Goodbye Lenin! only).

Res (Event Resolution): When the Event fully resolves after triggering, either Immediate, Next Round, Next Turn, Next SC, Next PS. An asterisk (*) is added if there are conditions and/or restrictions.

Dur (Event Duration): The Duration of the Event, either Round (R), Turn (T), Play In Front Of Player (PIFOP), Game (G), Not Applicable (N/A).

Country (Country or Region): The country or region this Event refers to, either BA (Balkans = BG & RO), BU (Bulgaria), EE (Eastern Europe = CSSR & DDR & HU & PL), CSSR (Czechoslovakia), DDR (East Germany), HU (Hungary), PL (Poland), RO (Romania).

Space (Space Type): Effected space type(s) or a specific city/cities. When there is overlap only the space type(s). Space types: Bureaucrat (B), Church (C), Elite (E), Intellectual (I), Minority (M), Student (S), Worker (W).

SP (Support Point Change): Does this Event cause the addition or removal of Support Points? Yes(*) (Condition*: SP presence & SP control & PS outcome) / No / Possible (Roll/Draw/Choice/Power/...).

SC (Support Check): Does this Event allow and Support Check? Yes(*) (Condition*: SP presence & SP control & PS outcome) / No / Possible (Roll/Draw/Choice/Power/...).

VP (Victory Point): Does this Event give Victory Points? Yes(*) (Condition*: SP presence & SP control & PS outcome) / No / Possible (Roll/Draw/Choice/Power/...).

PS (Power Struggle): Does this Event effect a Power Struggle? Yes(*) (Condition*: SP presence & SP control & PS outcome) / No / Possible (Roll/Draw/Choice/Power/...).

TST (Tiananmen Square Track): Does this Event effect the Tiananmen Square Track? Yes(*) (Condition*: SP presence & SP control & PS outcome) / No / Possible (Roll/Draw/Choice/Power/...).

HS (Hand Size): Does this Event effect Hand Size? Yes(*) (Condition*: SP presence & SP control & PS outcome) / No / Possible (Roll/Draw/Choice/Power/...).
