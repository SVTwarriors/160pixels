Card Text: +2 VP. The Democratic Player may place SPs or make Support Checks using the Ops value of this card if "This Wall Must Go!" has been played as an Event successfully.

   

Democrat play: The Democrat should always trigger this event as it doesn’t have a downside. If the The Wall Must Go! event hasn’t yet been triggered an there is a chance it will in the next Action Round(s) or Turn(s) the Democrat should when possible try to hang on to Kohl Proposes Reunification.

 

   

Communist play: An annoying event for the Communist player once the The Wall Must Go! event has been triggerd he should try to play Kohl Proposes Reunification before The Wall Must Go! if his position in East Germany is weak. If it is too late for that this card is a good TST candidate.
