Card Text: -2 VP Remove any Democratic SPs in Razgrad.

   

Democrat play: The Bulgarian Turks Expelled event is easy to trigger for the Democrat player if he hasn’t put any SPs in the “Razgrad” space through either the Sajudis event or the The Baltic Way event. This is rare however. When triggering Bulgarian Turks Expelled event the Democrat player should consider how triggering the event would weaken the Ruse space in regards to Communist Support Checks if he has Ops there (e.g. from the Eco-glasnost event), what the odds are of triggering the The Baltic Way event later on, the Bulgaria scoring event and if the 2 VP can help the Communist player reach 20 VP or help him with the 40th Anniversary Celebration event or the New Year’s Eve Party event.

 

   

Communist play: The Bulgarian Turks Expelled event is best triggered by the Communist player when there isn’t a high chance of the Democrat player quickly repairing the damage with the The Baltic Way event. The things to consider are the same as for the Democrat player. Late Year The Bulgarian Turks Expelled event can be used to try to trick the Democrat player in triggering the Breakaway Baltic Republics early so the Communist player can trigger the Kremlin Coup! event. Mostly the Communist player will need to play the Bulgarian Turks Expelled card for 3 Ops however.
