Card Text: Remove 2 opponent SPs and make a Support Check using the Ops value of this card. May not be played as an Event after Breakaway Baltic Republics.

   

Democrat play: Early on in the game the Democrat player should be able to find a nice target for the Gorbachev Charms the West event’s Support Check after it’s 2 SP removal unless the Communist player has done some serious overprotecting. Considering an average Support Check roll the event should normally result in a bigger SP change than by placing SP with the 4 Ops but rarely if the Democrat player already has a good Support Check modifier on a target (and preferably on a second backup target) the 4 Ops can better be used for two Support Checks. The Democrat player should not hold the Gorbachev Charms the West card into the next Turn 3 if the Deutsche Marks card is still in the draw deck, just like all other 4 Ops card in fact throughout the game. This is particularly important going into Turn 3 and Turn 7 because of card counting. Reminder: The Gorbachev Charms the West event is no longer possible after the Breakaway Baltic Republics event has been triggered and after this can no longer be used to counter the Ligachev event.

 

   

Communist play: Normally the Communist player will play the Gorbachev Charms the West event. Early Year before the Democrat player has spread too much the SP removal and Support Check of the event can be used independently to deny the Democrat player access in Czechoslovakia, in East Germany or in any Intellectual space(s). This of course requires the Communist player has already set up a Support Check with a sufficient modifier without making use of the Gorbachev Charms the West event’s SP removal as not to waste it. Reminder: The Gorbachev Charms the West event is no longer possible after the Breakaway Baltic Republics event has been triggered and after this can no longer be used to counter the Ligachev event.
