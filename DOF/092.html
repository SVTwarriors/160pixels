Card Text: Choose any Orthodox Church space. Replace all Democratic SPs with Communist SPs.

   

Democrat play: Normally the Democrat player, knowing this card is in the Late Year deck, has refrained from moving into Orthodox Church spaces. If this was the case this game then Betrayal is easy to trigger for the 3 Ops. If the Democrat player is in control of an Orthodox Church space in Bulgaria and/or Romania he should TST it.

 

   

Communist play: If there is an uncontrolled Orthodox Church space the Ops should be (when sensible) used to take control of it, because once Betrayal is out of the game the Democrat player will be able to move into the Orthodox Church space.
