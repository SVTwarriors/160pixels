Card Text: Remove four Democratic SPs, no more than 2 per space.

   

Democrat play: The Democrat player should never trigger the State Run Media event and should always try to TST it (preferably in Turn 3). The Democrat player cannot repair the SP damage done by the State Run Media event with the 2 Ops. The State Run Media event can also be used for access denial in Hungary and Czechoslovakia in particular. This effect is especially pronounced in in Early Year when the Democrat player sometimes has only a tenuous foothold in some countries. Halfway the game access denial by the Communist player with the State Run Media event is often no longer possible. Aside from Dissidents Arrested, the State Run Media event is a reason for the Democrat player to place 3 SPs in intellectual spaces after they have been protected by Helsinki Final Act to avoid being kicked out completely by the Communist player.

 

   

Communist play: The State Run Media event can be used to deny the Democrat player access to entire countries/regions Early Year or Student spaces and Intellectual spaces after they have been “protected” against Support Checks with the Helsinki Final Act event. Otherwise the State Run Media event can be triggered the Action Round after the Democrat player announces he will play a low Ops Card, a card he will play for the event or a Scoring card the next Action Round because of the Stasi event. Hereby the State Run Media event can create Support Check targets or effect a countries Power struggle and Scoring. The best way to use the State Run Media event is on the last Action Round of a Turn by combining it with the Honecker event or multiple Democrat discards (e.g. the Deutsche Marks event, the Brought in for Questioning event, the Common European Home event and to a lesser extent the Inflationary Currency event). This can allow for a Action Round 1 Support Check with an advantageous modifier the next Turn or drastically influence Final Scoring on Turn 10. Otherwise the Communist player should play the State Run Media card for the Ops keeping it in the game and hoping it returns to his hand when he can use it for any of the above.
